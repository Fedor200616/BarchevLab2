# Руководство пользователя

---

## Назначение программы

Программа предназначена для:
- поиска информации о студентах;
- вывода данных в файл.

Для работы используются два входных файла:
- файл с данными студентов;
- файл с паролями.

---

## Главное меню

После запуска программы необходимо:
1. Выбрать файл с данными студентов.
2. Выбрать файл с паролями.

После выбора файлов доступны следующие действия:
- Поиск данных пользователя;
- Полный вывод данных в файл.

---

## Поиск данных пользователя

В данном режиме пользователь может выполнить поиск по следующим полям:
- имя;
- фамилия;
- группа;
- номер.

### Особенности работы:
- Под строкой ввода отображается количество найденных совпадений;
- Если найдено менее 12 записей, они выводятся на экран;
- Для просмотра результатов нажмите клавишу Enter (при наличии совпадений);
- Для возврата в главное меню нажмите клавишу Esc.

### Пример окна поиска:

Введите строку для поиска (по фамилии, группе или номеру):
С
Найдено совпадений: 3
Имя | Группа | Номер | Пароль
Сидоров | М7О-205БВ-24 | 12 | userE5f6_12
Смирнов | М7О-207БВ-24 | 7 | userJ9k0_7
Соколов | М7О-210БВ-24 | 10 | userP5q6_

Нажмите Enter для просмотра результатов
Нажмите Esc для выхода в главное меню


---

## Полный вывод данных в файл

В данном режиме осуществляется экспорт данных студентов в файл.

### Доступные параметры:
1. Выбор формата файла:
   - txt
   - csv

2. Выбор типа данных:
   - все записи;
   - только полностью заполненные записи.

3. Выбор разделителя (только для формата txt):
   - |
   - ,

4. Вывод результата  
   После выбора данного пункта открывается окно для указания пути и имени файла.

5. Возврат в главное меню.

---

## Управление

Навигация по меню осуществляется с клавиатуры:
- стрелка вверх / вниз — перемещение по пунктам меню;
- стрелка влево / вправо — изменение параметров;
- Enter — выбор пункта;
- Esc — возврат в предыдущее меню.
